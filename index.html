Visor

Visor is a live coding environment for creative coding of graphics in audiovisual performances.
Live coding is performed in the Ruby language and graphical capabilities are provided by Processing.

Downloads

Visor is currently in alpha and subject to change. Testing, feedback, and critique are welcomed!

MIDI
na1, na2, na3, na4 na1_down, na2_down, na3_down, na4_down na1_up, na2_up, na3_up, na4_up nb1, nb2, nb3, nb4 nb1_down, nb2_down, nb3_down, nb4_down nb1_up, nb2_up, nb3_up, nb4_up ka1, ka2, ka3 kb1, kb2, kb3 kc1, kc2, kc3 s1, s2, s3

Please note
Currently, the Novation Launch Control XL is the only supported controller and partially at that - full support for generic controllers is coming soon.
n* maps to a note button, k* maps to a knob control, s* maps to a slider control.
